# CRYPTO PLAYGROUND

## Getting Started

### Pre-requisites

- NodeJS: v20
- Yarn: v4

### Setup

```
corepack enable
```

```
yarn install
```

### Run dev mode

```
yarn dev
```

### Test

```
yarn test
```

### Cleanup

```
yarn cleanup
```

#### Cleanup coverage

```
yarn cleanup:coverage
```

#### Cleanup node_modules

```
yarn cleanup:nm
```
